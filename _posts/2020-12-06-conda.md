# How to use Conda
## Packages

+ List installed packages

	```conda list```

+ Install a package / packaages

  ```conda install pandas```

  ```conda install pandas numpy tensorflow```

  ```conda install jupyter notebook```

  ```conda install nb_conda```

+ Update package / packages

  ```conda update pandas```

  ```conda update--all```

+ Remove a package

  ```conda remove pandas```

+ Search packages

  ```conda search pandas```

  ```conda search *pandas*```
