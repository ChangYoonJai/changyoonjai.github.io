## Flutter 시작

### Flutter 설치
```git clone https://github.com/flutter/flutter.git -b stable```

#### Flutter 설치 확인
```flutter doctor```

### Flutter Project 생성 A
```flutter create <app_name>```

### Flutter Project 생성 B (VS Code Flutter 확장)
```Flutter: New Project > app```

### Flutter Project 실행
```F5``` or ```ctrl + F5```
